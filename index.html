On this chance, we will learn about matrix

Matrix is a set of numbers arranged in rows or columns or both, placed in brackets or square brackets. Now, the numbers that make up a matrices are called matrices elements. The matricx is used to simplify data delivery, so that it can be easily processed.

Besides that, there are types of matrices that you should know, including:

  1. Zero Matrix

Zero matrix is a matrix in which all of the entries are “0”.

  1. Row Matrix

Row matrix is a matrix that only have one Row.

  1. Column Matrix

Column matrix is a matrix that only have one column.

  1. Square Matrix

An n × n matrix is said to be a square matrix of order n. In other words when the number of rows and the number of columns in the matrix are equal then the matrix is called square matrix.

  1. Identity Matrix

An identity matrix is a given square matrix of any order which contains on its main diagonal elements with value of one, while the rest of the matrix elements are equal to zero.
